# bot-罢了

**bot-罢了**是基于 nonebot + go-cqhttp 开发的qq机器人.

## 各机器人的作用

### bot-01

向机器人私聊发送 `/echo $content` 

机器人私聊回复 `$content`

### bot-02

向机器人私聊发送 `天气 $city`

机器人私聊回复 `$city 的天气是：`

### bot-03

向机器人私聊 或者 在群里发送在 `data.txt` 中指定的字段

机器人私聊 与在群里发送 同一字段
